🔒 隐私政策

最后更新： 2025年6月30日

简要说明： Gemini Web Summarizer 扩展致力于保护您的隐私。我们只收集必要的信息来提供服务，不会出售或分享您的个人数据。

1. 信息收集

1.1 我们收集的信息

  • API密钥： 您提供的Google Gemini API密钥，仅存储在您的本地设备上
  • 网页内容： 您选择总结的网页文本内容
  • 使用统计： 基本的功能使用次数（不包含个人身份信息）
  • 设置偏好： 您的扩展配置和偏好设置

1.2 我们不收集的信息

  • 个人身份信息（姓名、邮箱、电话等）
  • 浏览历史记录
  • 密码或其他敏感凭据
  • 位置信息

2. 信息使用

2.1 使用目的

  • 服务提供： 使用您的API密钥调用Google Gemini服务生成总结和流程图
  • 功能改进： 分析使用统计以改进扩展功能
  • 技术支持： 诊断和解决技术问题

2.2 数据处理原则

  • 最小化原则： 只收集和处理必要的数据
  • 目的限制： 数据仅用于明确说明的目的
  • 透明度： 清楚说明数据的收集和使用方式

3. 数据存储和安全

3.1 存储位置

  • 本地存储： API密钥和设置存储在您的浏览器本地
  • 临时处理： 网页内容仅在处理过程中临时使用
  • 不存储内容： 我们不永久存储您的网页内容或生成的总结

3.2 安全措施

  • 加密传输： 所有API调用使用HTTPS加密
  • 安全存储： 使用浏览器的安全存储机制
  • 访问控制： 只有您的扩展可以访问存储的数据
重要提醒： 您的API密钥具有访问Google Gemini服务的权限，请妥善保管，不要与他人分享。

4. 第三方服务

4.1 Google Gemini API

  • 我们使用Google Gemini API来生成内容总结和流程图
  • 发送给API的数据受Google的隐私政策约束
  • 请查阅 Google隐私政策 了解详情

4.2 数据传输

  • 网页内容会发送给Google Gemini API进行处理
  • 传输过程使用HTTPS加密
  • 我们不会将数据发送给其他第三方服务

5. 用户权利

5.1 您的控制权

  • 数据访问： 您可以随时查看存储的设置和偏好
  • 数据修改： 您可以随时修改或删除设置
  • 数据删除： 卸载扩展将删除所有本地存储的数据
  • 服务停用： 您可以随时停止使用扩展

5.2 数据可携性

  • 您可以导出扩展设置
  • 设置可以在不同设备间同步（通过浏览器同步功能）

6. 数据保留

  • 设置数据： 保留直到您删除或卸载扩展
  • 使用统计： 本地保留30天，自动清理
  • 临时数据： 处理完成后立即删除

7. 儿童隐私

本扩展不专门针对13岁以下儿童设计。如果我们发现收集了儿童的个人信息，将立即删除相关数据。

8. 政策更新

  • 我们可能会不时更新此隐私政策
  • 重大变更将通过扩展更新通知用户
  • 建议定期查看最新版本的隐私政策

9. 法律依据

我们处理您的数据基于以下法律依据：

  • 合同履行： 为您提供扩展服务
  • 合法利益： 改进产品功能和用户体验
  • 用户同意： 您使用扩展即表示同意此政策

10. 联系我们

如果您对此隐私政策有任何疑问或建议，请通过以下方式联系我们：

  • GitHub Issues： 在项目仓库中提交问题
  • 扩展商店： 通过Chrome/Edge扩展商店反馈

响应时间： 我们将在7个工作日内回复您的询问。

感谢您信任 Gemini Web Summarizer！

我们承诺持续改进隐私保护措施，为您提供安全、可靠的服务。
